JCP - Java Community Process

  • Formalized mechanism for developing Java specifications
  • Not standards organization / legal entity - dominated by Oracle
  • JSR - Java Specification Request:
    • Specification, Reference Implementation & TCK
    • TCK - Technology Compatibility Kit
  • Criticized as "rubber stamp organization", but:
    • Apache vs. Sun on getting access to the TCK
    • JCP EC rejects JSR 376 (Jigsaw)
  • Three Java platforms: Java ME, Java SE and Java EE ⇒ EE4J
